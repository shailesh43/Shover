[Shover](/) is a modern UI library built on top of **shadcn/ui**, designed to help developers build sleek, animated, and production-ready interfaces with less effort.

## Why Shover?

Building beautiful interfaces shouldn't mean stitching together dozens of libraries or rewriting the same components every project.

Shover combines reusable components, smooth animations, thoughtfully designed blocks, and modern development practices into one cohesive library—helping you build polished interfaces faster while keeping full control over your code.

## Open Source & Community-Driven

Shover is completely open source and built in public. The goal is to build a collection that's practical, well-crafted, and genuinely useful for the developer community.

## Design Philosophy

Shover is built around a simple idea: **copy, customize, and make it yours.**

Every component is designed to be production-ready, easy to understand, and flexible enough to fit your own design system. Rather than hiding implementation behind abstractions, Shover encourages developers to own the code they ship, making customization straightforward and future maintenance easier.
